# Descrição do software
Este projeto implementa um sistema simples de gerenciamento de freelancers, contratantes e projetos, desenvolvido em Python. Ele permite cadastrar profissionais, registrar empresas contratantes e criar projetos associados a elas. Também possibilita vincular freelancers aos projetos conforme suas competências.


O sistema conta com classes bem definidas: Pessoa, Freelancer, Contratante e Projeto, cada uma contribuindo com atributos e comportamentos específicos. A classe Gerenciamento centraliza e organiza os cadastros, controlando a inserção de freelancers e projetos, além de garantir que não ocorram duplicidades através de tratamento de exceções. Também é possível associar freelancers a projetos, permitindo visualizar equipes formadas para cada demanda.



# Informações
### Estrutura do Projeto

O arquivo freelas.py contém as classes de domínio, enquanto o main.py demonstra o uso do sistema com cadastros automáticos e sem necessidade de entrada do usuário

### Como executar o codigo

Carregue os arquivos main.py e freelas.py na mesma pasta, e através do terminal do sistema ou IDE, execute o arquivo main.py

Ao executar o arquivo main.py, o programa irá exibir:

- O cadastro dos freelancers
- O cadastro de contratante
- A criação do projeto
- A associação de freelancers aos projetos

### Personalização

Modificando as listas e as chamadas do main.py o usuário pode:
- Adicionar novos freelancers
- Criar novos projetos
- Associar profissionais a projetos
- Expandir o sistema adicionando menus ou arquivos de salvamento
